Transferring your Course Schedule to other Calendar Applications


From Minerva, McGill Enhanced allows you to download your course schedule as an ICS file which can then be imported into Google Calendar, Apple iCal, Microsoft Outlook, or other calendar apps that can import ICS files.


Schedule Download

Simply go to Minerva -> Student Menu -> Registration Menu -> Step 4. View Student Schedule by Course Section, then click the "Download Course Schedule as ICS file!" button and McGill Enhanced will download your schedule as an ICS file.




Importing ICS file

To import the downloaded ICS file into Google Calendar, for example, select import calendar under the Other calendars menu, then choose the .ICS file and choose the calendar you wish to import it into, then click Import.
